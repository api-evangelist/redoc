New release: replay@0.20.0 · Date: 2026-05-11 · 0 features · 8 fixes

Fixes:
• Updated `@redocly/openapi-core` and `@redocly/respect-core` to version `2.30.3`.
• Updated `@redocly/openapi-core` and `@redocly/respect-core` to version `2.30.1`.
• Fixed security vulnerability `CVE-2026-41650` by upgrading `fast-xml-parser` to version `5.7.1`.
• Updated `@redocly/openapi-core` and `@redocly/respect-core` to version `2.30.2`.
• Fixed an issue in Replay where stale credentials persisted in the active environment across page reloads, and caused intermittent "JWT is expired" errors.
• Updated `@redocly/openapi-core` to version `2.29.1`.
• Fixed an issue in Replay where the request body with invalid JSON input grew after restarts, leading to decreased performance.
• Improved typing performance in the request URL field.
